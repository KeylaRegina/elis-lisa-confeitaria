# 🍰 Elis & Lisa Confeitaria

## 📌 Sobre o projeto

Este projeto foi desenvolvido como uma aplicação real para uma confeiteira, com o objetivo de criar uma presença online para divulgação dos seus produtos.

A aplicação apresenta uma vitrine de bolos personalizados, utilizando imagens reais dos produtos e identidade visual baseada na marca da cliente.

Além disso, o projeto também teve como finalidade o aprimoramento das minhas habilidades em desenvolvimento front-end.

---

## 🎯 Objetivo

* Desenvolver uma landing page funcional para uma cliente real
* Criar uma vitrine digital para divulgação de produtos
* Praticar desenvolvimento front-end com foco em projetos reais

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 💻 Funcionalidades

* Página inicial com destaque visual (hero section)
* Navegação entre seções (Início, Bolos, Doces, Mini Festa, Contato)
* Galeria com imagens reais dos produtos
* Layout responsivo (mobile e desktop)
* Formulário de contato
* Integração com WhatsApp para pedidos

---

## 🎨 Identidade visual

A interface foi desenvolvida com base na identidade da confeiteira, incluindo:

* Uso de cores suaves e temáticas
* Imagens reais dos produtos
* Caricatura personalizada baseada na cliente

---

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em diferentes dispositivos, garantindo uma boa experiência tanto em celulares quanto em desktops.
